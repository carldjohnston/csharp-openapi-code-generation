# Agl.Cdr.Energy - ASP.NET Core 3.1 Server

Consumer Data Right end points and payloads for the Energy sector

## Run

Linux/OS X:

```
sh build.sh
```

Windows:

```
build.bat
```
## Run in Docker

```
cd src/Agl.Cdr.Energy
docker build -t agl.cdr.energy .
docker run -p 5000:8080 agl.cdr.energy
```
